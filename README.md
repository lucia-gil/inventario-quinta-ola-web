# Proyecto IWEB
# Entregable 2 - Inventario Quinta Ola (HTML)

## Descripción
Este repositorio contiene el desarrollo del Entregable 2 del proyecto "Inventario Quinta Ola", enfocado en la implementación completa en HTML de las pantallas principales del sistema.

## Módulos implementados

### Seguridad y accesos
- Login
- Registro
- Panel de aprobación de usuarios
- Asignación de roles
- Dashboard SuperAdmin

### Flujo del solicitante
- Catálogo de materiales
- Detalle de material
- Carrito / Solicitud

### Aprobación y coordinación
- Bandeja de solicitudes
- Modal de rechazo
- Detalle de solicitud
- Historial y trazabilidad

### Inventario y depósito
- Panel de inventario
- Registro de entradas
- Panel de despacho

### Reportes
- Dashboard con métricas
- Exportación simulada

### Elementos globales
- Navbar
- Notificaciones
- Perfil de usuario
- Estados vacíos
- Validaciones

## Tecnologías
- HTML5
- CSS3
- JavaScript (básico)

## Objetivo
Simular la interfaz completa del sistema sin backend, priorizando estructura, diseño y experiencia de usuario.
